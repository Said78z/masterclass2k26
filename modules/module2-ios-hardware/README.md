# Module 2 : Hardware & IoT (Proof of Physical Work)

Apprenez à simuler des capteurs physiques pour les réseaux DePIN qui exigent des preuves de travail physique (WeatherXM, Helium).

## Contenu
- **TP1 ESP32 Sim** : Code Arduino/ESP32 pour simuler une station météo avec signature cryptographique.

## Objectifs
1.  Gérer des microcontrôleurs pour le Web3.
2.  Optimiser la consommation d'énergie (Deep Sleep).

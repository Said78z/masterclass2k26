# Module 1 : L'Usine à Nœuds DePIN

Ce module se concentre sur l'automatisation de l'infrastructure pour les réseaux DePIN (Grass, Titan Network).

## Contenu
- **TP1 Infrastructure** : Provisioning de serveurs via Terraform et configuration Docker via Ansible.
- **TP2 Deployment** : Utilisation de Docker Compose avec un sidecar `gluetun` pour masquer l'IP datacenter.

## Objectifs
1.  Comprendre l'importance de la réputation IP.
2.  Automatiser la gestion de centaines de nœuds sans intervention humaine.
